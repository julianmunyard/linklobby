  <how-to-verify>
**Start the dev server:** `npm run dev`
**Open:** http://localhost:3000/editor

### 1. Dropdown Card (5 min)
- [ ] Add a Dropdown from card picker
- [ ] Verify it appears collapsed with "Show more" text
- [ ] Click to expand - animation should be smooth
- [ ] Edit header text, expand text, collapse text in property editor
- [ ] Verify text changes appear in dropdown

### 2. Nested Drag-and-Drop (5 min)
- [ ] Expand a dropdown
- [ ] Drag a card INTO the dropdown - should show ring highlight on hover
- [ ] Card should appear inside dropdown after drop
- [ ] Drag a card OUT of dropdown to main canvas
- [ ] Card should return to main canvas
- [ ] Try dragging a dropdown into another dropdown - should NOT work

### 3. Snake Game (3 min)
- [ ] Add a Game card from picker
- [ ] Verify default is Snake
- [ ] Click Play - game should start
- [ ] Test arrow keys / WASD controls
- [ ] Test swipe controls on mobile
- [ ] Let snake hit wall - Game Over should show
- [ ] Click Play Again

### 4. Breakout Game (3 min)
- [ ] Change game type to Breakout in property editor
- [ ] Click Play
- [ ] Move mouse/touch to control paddle
- [ ] Ball should bounce and break bricks
- [ ] Score should increase per brick
- [ ] Let ball fall below paddle - Game Over

### 5. Flappy Game (3 min)
- [ ] Change game type to Flappy
- [ ] Click Play
- [ ] Click/tap to flap
- [ ] Navigate through pipes
- [ ] Score should increase per pipe passed
- [ ] Hit pipe or ground - Game Over

### 6. Multi-Select Desktop (5 min)
- [ ] Click and drag to draw selection box
- [ ] Cards within box should get white ring selection
- [ ] Shift+click to add/remove individual cards
- [ ] Selection toolbar should appear at bottom
- [ ] Click "Group into Dropdown" - new dropdown with cards
- [ ] Select more cards, use "Move to Dropdown" submenu
- [ ] Select cards, click "Delete All" - should show confirmation
- [ ] Click X to clear selection

### 7. Multi-Select Mobile (5 min)
- [ ] Open on mobile (or use DevTools responsive mode)
- [ ] Find and tap "Select" button
- [ ] Cards should show checkbox overlays
- [ ] Tap checkboxes to select cards
- [ ] Selection count should update
- [ ] Bulk actions should work
- [ ] Tap "Done" or "Exit Select" to leave mode

### 8. Edge Cases (3 min)
- [ ] Game card in editor shows preview only (no gameplay)
- [ ] Games adapt to card size (big vs small)
- [ ] Dropdown badge shows correct child count
- [ ] Empty dropdown shows "Drag cards here" message
